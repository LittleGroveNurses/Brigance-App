# briganceActivities.js

This file needs to be uploaded to your GitHub repository alongside brigance_app.html.

The content is the complete briganceActivities.js file provided in your session.
It defines:
  - briganceActivities (393 activities across 34 domains)
  - briganceRouter (IED III code → domain key mapping)
  - briganceDomainIndex (domain metadata)

For GitHub Pages, the file must be named exactly: briganceActivities.js

After uploading, add this line inside the <head> of brigance_app.html:
  <script src="briganceActivities.js"></script>

The app will automatically detect the loaded data and activate the Activities tab.
